# 🚦 Smart Traffic & Car System

Bu proje, temel C programlama prensipleri kullanılarak geliştirilmiş; trafik yönetimi ve araç maliyet analizlerini içeren çok modüllü bir konsol uygulamasıdır.

## 🛠️ Özellikler

* **Akıllı Kavşak Yönetimi**: Araç yoğunluğuna göre trafik ışığı süre tavsiyesi verir.
* **Yol Bilgisayarı**: Yakıt tüketimi ve maliyet analizi yapar.
* **Hız Koridoru Radar Kontrolü**: Ortalama hız hesaplayıp ceza kontrolü yapar.

## 🚀 Nasıl Çalıştırılır?

1. Terminali açın.
2. `gcc trafik.c -o trafik.exe` komutuyla derleyin.
3. `./trafik.exe` komutuyla çalıştırın.
